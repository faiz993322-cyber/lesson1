Goal

The goal is to get introduced to basic elements related to text - paragraphs, headings, and quotations. Also, understand various formattings by creating a webpage of your own.

Getting started

Getting started

  1. HTML <body> tag defines the main content of an HTML document which displays on the browser. It can contain text content, lists, paragraphs, headings, images, tables, links, videos, etc.
  2. You need to complete this activity in the same way we did in the classroom.

Example: A small paragraph + code

Below is an example paragraph and the HTML code used to create it. You will type similar text in your project.

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>My Webpage</title>
  </head>
  <body>
    <h1>Welcome to my page</h1>
    <p>This is a short paragraph about my topic.</p>
  </body>
</html>

Tip: Replace the sample text with your own story. Leave space for handwriting if your teacher asks.
